# 🔍 日常的微歧視：校園與職場的隱性偏見偵查線

本專案積極響應聯合國永續發展目標 **SDG 5 性別平權** 與 **SDG 10 減少不平等**，是一套專為辨識日常言語盲點、提供法律自衛導引而開發的輕量化數位倡議互動平台。

---

## 🕊️ 核心模組介紹

本平台打破傳統教條式宣導的死板框架，由以下五大靜態與互動頁面組成[cite: 4]：

1. **倡議主首頁 (`index.html`)**
   整合平台宗旨與模組導覽，以直覺的幾何方塊引導使用者開啟反思行動[cite: 4]。
2. **微光匿名故事樹洞 (`anonymous.html`)**
   核心主打功能[cite: 4]。為面對結構不對等、不敢走正式申訴管道的個體打造[cite: 4]。提供完全去識別化、免除後端洩漏風險的本地端安全發聲牆，凝聚翻轉偏見的改變力量[cite: 4]。
3. **偏見偵查小隊 (`game1.html`)**
   輕量化對話解謎測驗[cite: 2]。劃分「微光職場」與「校園象牙塔」事件簿，揭示包裝在善意與常理下的無形微歧視[cite: 2]。內建選項標準亂數洗牌機制，杜絕死記答案[cite: 2]。
4. **隱性偏見天秤大分類 (`game2.html`)**
   盲測抉擇遊戲[cite: 3]。將日常字卡歸類為「真心讚美」或「潛在標籤」[cite: 3]。一旦做出帶有潛意識盲點的錯誤歸類，重力天秤將當場失衡歪斜，給予直覺的視覺化審查規訓衝擊[cite: 3]。
5. **臺灣性平三法求助指南 (`law.html`)**
   法律保障安全網[cite: 5]。將艱澀條文轉化為直覺的「自我防衛 3 步驟圖卡」，提供一鍵直撥官方諮詢專線與中央直屬民意信箱連結，全力反制組織吃案盲區[cite: 5]。

## 🛠️ 前端技術亮點

本專案全面落實 **HTML / CSS / JavaScript 三層分離標準架構**，兼顧極佳的 UI 視覺質感與 UX 排版流暢度：

### 🟢 HTML：結構與排版
* 建立倡議首頁、故事樹洞牆、偏見偵查小隊、天秤大分類與法規求助指南共五頁的排版結構。

### 🔵 CSS：視覺風格與原子化佈局
* **原子化 CSS 佈局**：全面導入 **Tailwind CSS 框架**，利用幾何網格（`grid`）快速佈署跨裝置相容的響應式網頁排版[cite: 4, 5]，並透過 `sticky` 黏性定位鎖定組件行為[cite: 1, 5]。
* **物理天秤失衡機制**：利用中軸控制（`transform-origin`）搭配自訂**貝茲曲線（`cubic-bezier`）**，平滑模擬偏見失衡時重力砸下的物理加速感[cite: 3].。
* **微互動動畫**：利用 `opacity` 與 `scale` 打造 0.4 秒流暢的減速淡入縮放轉場[cite: 2, 3]，並配合 Y 軸垂直位移設計首頁卡片的 3.5 秒無限循環微懸浮動態[cite: 4].。

### 🟡 JavaScript：動態行為大腦
* **安全隱性快取**：完全不依賴後端資料庫，純粹採用 `localStorage` 技術進行故事便利貼的本地存取與去識別化安全隔離，達成零後端洩漏風險[cite: 1]。
* **Fisher-Yates 亂數演算法**：針對情境選項與天秤盲測字卡進行獨立的隨機洗牌，打破僵化的死背模式[cite: 2]。
* **數據模型與動態改寫**：即時動態改寫 DOM 樹樣式（`style.transform`）驅動天秤歪斜[cite: 3]，並依據玩家抉擇進行偏見指數得分統計與綜合判定報告生成[cite: 2, 3]。
