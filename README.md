# 🔍 日常的微歧視：校園與職場的隱性偏見偵查線

本專案積極響應聯合國永續發展目標 **SDG 5 性別平權** 與 **SDG 10 減少不平等**，是一套專為辨識日常言語盲點、提供法律自衛導引而開發的輕量化數位倡議互動平台[cite: 4, 5]。

## 🕊️ 核心網頁架構

本平台打破傳統教條式宣導的死板框架，由以下五大網頁組成。所有頁面均採用單一檔案（Single-file）架構，將網頁骨架、視覺樣式與動態邏輯高度整合：

1. **倡議主首頁 (`index.html`)**
   全站的入口導覽核心。整合平台宗旨，利用響應式幾何方塊與微漂浮動態，引導使用者自由選擇開啟反思行動。
2. **微光匿名故事樹洞 (`anonymous.html`)**
   專為面對權力不對等、不敢走正式申訴管道的個體所量身打造[cite: 1, 4]。提供完全去識別化、免除後端洩漏風險的本地端安全發聲牆，抒發委屈並凝聚翻轉偏見的改變力量[cite: 1, 4]。
3. **偏見偵查小隊 (`game1.html`)**
   輕量化情境對話解謎測驗[cite: 2, 4]。劃分「微光職場」與「校園象牙塔」等多個迷惑性事件簿，帶領大眾主動拆解包裝在善意關懷、傳統讚美底下的「隱性偏見」與「微歧視」[cite: 2]。
4. **隱性偏見天秤大分類 (`game2.html`)**
   盲測抉擇遊戲[cite: 3, 4]。從海量題庫中隨機抽題，要求使用者運用平權直覺將日常對話字卡歸類為「真心讚美」或「潛在標籤」[cite: 3]。
5. **臺灣性平三法求助指南 (`law.html`)**
   法律保障安全網[cite: 4, 5]。整合臺灣核心「性平三法」法規，將艱澀條文轉化為直覺的「自我防衛 3 步驟圖卡」，並提供一鍵直撥官方諮詢專線與中央直屬民意信箱連結[cite: 5]。

## 🛠️ 前端技術亮點與實作原理

網頁雖然沒有拆分複雜的外部後端，但底層深度運用了現代網頁開發的核心技術，兼顧極佳的 UI 視覺質感與 UX 操作流暢度：

### 🟢 網頁佈局：Tailwind CSS 原子化與響應式格線
* **原子化佈局**：全面導入 Tailwind CSS 框架，免去繁瑣的命名與跨檔案切換，直接在 HTML 標籤內建構樣式[cite: 4, 5]。
* **響應式跨裝置相容**：利用幾何網格（如 `grid-cols-1 lg:grid-cols-3`）與彈性盒（`flex`），極速實現智慧型手機與電腦端無縫相容的跨裝置完美排版[cite: 4, 5]。
* **組件行為鎖定**：在發布表單與導覽組件套用 `sticky top-6 h-fit` 與 `relative z-10` 特效，確保使用者不論在牆上滑動多深，操作面板都能牢牢固定於視窗頂端[cite: 1, 5]。

### 🔵 視覺特效：內嵌式 CSS 三大核心動畫
* **轉場微互動（圖片一 `image_cd9b67.png`）**：在 `<style>` 中自訂 `@keyframes fadeIn` 關鍵影格，設定元素登場在 0.4 秒內完成，並伴隨 2% 的細微物理縮放差（`scale(0.98)` 至 `1`），讓解析字卡如同呼吸般自然浮現[cite: 2, 3]。
* **物理天秤失衡機制（圖片二 `image_cd9ae7.png`）**：設定橫梁 `.balance-beam` 的旋轉動作在 0.6 秒內完成，精確固定旋轉軸心（`transform-origin: center 20px`）並導入自訂**貝茲曲線（`cubic-bezier(0.4, 0, 0.2, 1)`）**，完美模擬偏見失衡時重力砸下的真實物理加速感[cite: 3]。
* **全站視覺漂浮動態（圖片三 `image_cd9804.png`）**：建立 `float` 循環動畫，利用 `translateY(-10px)` 垂直位移與 `ease-in-out` 曲線，設定長達 3.5 秒的極慢速無限循環（`infinite`），賦予首頁吉祥物與卡片輕量化的懸浮生命力[cite: 4]。

### 🟡 動態大腦：JavaScript 邏輯功能運算
* **安全隱性快取**：完全不依賴後端資料庫，純粹採用 `localStorage` 技術進行故事便利貼的本地端存取、篩選與去識別化安全隔離，達成零後端洩漏風險[cite: 1]。
* **Fisher-Yates 隨機洗牌演算法**：針對偵查小隊的情境選項與天秤盲測的 20 題海量資料庫進行獨立的隨機抽題與亂數洗牌，打破僵化的死背模式[cite: 2, 3]。
* **數據模型與動態 DOM 控制**：當分類錯誤時，JavaScript 會即時計算並改寫 DOM 樹樣式（`style.transform = "rotate(12deg)"`）來控制天秤傾斜[cite: 3]；並在題組完結時，自動累加統計偏見指數得分，即時生成獲得認證稱號的綜合判定報告[cite: 2, 3]。
